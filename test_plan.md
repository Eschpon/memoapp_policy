# myplan 実機テスト項目

App Store審査提出前に、以下のテスト項目を実機で確認してください。

## 1. 起動・基本動作

- [ ] アプリが正常に起動する
- [ ] 起動後にクラッシュしない
- [ ] アプリアイコンが正しく表示される
- [ ] 3つのタブ（ホーム・検索・設定）が表示される
- [ ] 各タブをタップして画面が切り替わる

## 2. カレンダー表示

- [ ] 今月のカレンダーが正しく表示される
- [ ] 日付の配置が正しい（1日の曜日が合っている）
- [ ] 日曜日が赤色で表示される
- [ ] 土曜日が青色で表示される
- [ ] 祝日が赤色で表示される（例: 1/1, 2/11, 2/23）
- [ ] 「<」ボタンで前月に移動する
- [ ] 「>」ボタンで次月に移動する
- [ ] 年の表示をタップしてメニューが表示される
- [ ] メニューから別の年を選択して切り替わる

## 3. メモ機能

- [ ] 日付をタップして選択状態になる（背景色が変わる）
- [ ] 選択した日付のメモ欄が表示される
- [ ] メモを入力して「保存」ボタンで保存できる
- [ ] 保存後、カレンダー上にグレーの丸印が表示される
- [ ] 同じ日付を再度タップすると保存したメモが表示される
- [ ] メモを編集して上書き保存できる
- [ ] 「削除」ボタンでメモを削除できる
- [ ] 削除確認ダイアログが表示される
- [ ] 削除後、カレンダーの丸印が消える
- [ ] アプリを完全終了して再起動しても、メモが保持されている

## 4. カラーマーカー

- [ ] オレンジのチェックボックスをタップしてON/OFFできる
- [ ] 緑のチェックボックスをタップしてON/OFFできる
- [ ] 紫のチェックボックスをタップしてON/OFFできる
- [ ] マーカーONで保存後、カレンダーに色付き丸が表示される
- [ ] 3色すべてONで保存すると、3つの丸が並んで表示される
- [ ] マーカーなし＋メモありの場合、グレーの丸が表示される
- [ ] 別の日付を選択してから戻ると、マーカーの状態が正しく復元される

## 5. 検索機能

- [ ] 検索タブに切り替わる
- [ ] キーワードを入力すると検索結果が表示される
- [ ] 部分一致で検索できる（例: 「会議」で「会議資料」がヒット）
- [ ] 大文字/小文字を区別せず検索できる
- [ ] 一致しないキーワードで「一致するメモはありません」と表示される
- [ ] 検索結果に日付、カラーマーカー、テキストが表示される
- [ ] 検索結果を左スワイプして削除できる
- [ ] 「日付指定」チェックボックスがデフォルトOFFである
- [ ] 「日付指定」をONにすると日付範囲ピッカーが表示される
- [ ] 日付範囲を指定して検索結果が絞り込まれる
- [ ] キーワード＋日付範囲の組み合わせ検索ができる
- [ ] 日付範囲のみ（キーワード空欄）でも検索できる

## 6. 設定機能

- [ ] 設定タブに切り替わる
- [ ] ダークモードのトグルが表示される
- [ ] ダークモードをONにすると画面がダークモードになる
- [ ] ダークモードをOFFにするとライトモードに戻る
- [ ] 言語選択（日本語 / English）が表示される
- [ ] Englishを選択するとUI全体が英語に切り替わる
- [ ] 日本語に戻すとUIが日本語に切り替わる
- [ ] 設定がアプリ再起動後も保持される

## 7. 多言語対応

- [ ] 日本語モード: タブ名が「ホーム」「検索」「設定」
- [ ] 英語モード: タブ名が「Home」「Search」「Settings」
- [ ] 日本語モード: カレンダーの年月が「2026年2月」形式
- [ ] 英語モード: カレンダーの年月が「2026 Feb」形式
- [ ] 日本語モード: メモ欄が「メモ（2026-02-14）」形式
- [ ] 英語モード: メモ欄が「Memo (2026-02-14)」形式
- [ ] 日本語モード: 日付ピッカーが日本語表示
- [ ] 英語モード: 日付ピッカーが英語表示
- [ ] 検索画面のプレースホルダーが言語に合わせて変わる
- [ ] 削除確認ダイアログが言語に合わせて変わる

## 8. 画面回転・レイアウト

- [ ] 縦向きで正しく表示される
- [ ] 横向きでレイアウトが崩れない
- [ ] 各種iPhone画面サイズで表示が崩れない（SE, 標準, Plus/Max）

## 9. エッジケース

- [ ] メモ未入力で「保存」を押しても問題ない
- [ ] 非常に長いテキストを入力して保存・表示できる
- [ ] 月をまたいで連続して移動しても問題ない
- [ ] 2020年1月まで戻れる
- [ ] 2050年12月まで進める
- [ ] アプリをバックグラウンドに移して戻しても正常動作する

## 10. パフォーマンス

- [ ] 大量のメモ（50件以上）を保存しても動作が遅くならない
- [ ] 検索が即座に結果を返す
- [ ] 月の切り替えがスムーズ
